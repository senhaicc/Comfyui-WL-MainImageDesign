# 🎨 Comfyui-WL-MainImageDesign

**专业电商主图提示词生成器 - 摩托车骑行装备专版**

一款基于 ComfyUI 的 AI 电商主图设计插件，专为摩托车头盔、骑行服、护具等骑行装备定制，自动生成高转化率的电商主图提示词。

---

## ✨ 核心特点

- 🏍️ **摩托车装备专属** - 9种专业风格针对骑行装备深度优化
- 🎯 **高转化率设计** - 基于13张高转化主图深度分析的排版规范
- 🖼️ **场景融合** - 产品融入真实骑行场景，告别纯白底平铺
- 📐 **精准排版** - 支持1:1和3:4两种主流电商比例
- 🌍 **双语支持** - 中英文提示词自由切换
- 🔗 **参考图锁定** - 上传产品图自动锁定外观，重建专业场景

---

## 📦 安装方法

### 方式一：手动安装

1. 将整个文件夹复制到 ComfyUI 的 `custom_nodes` 目录：
```
ComfyUI/custom_nodes/Comfyui-WL-MainImageDesign/
├── __init__.py
├── prompt_nodes.py
├── prompts/
│   ├── system_prompt.txt
│   └── default_prompt.txt
└── README.md
```

2. 重启 ComfyUI

### 方式二：Git Clone

```bash
cd ComfyUI/custom_nodes
git clone https://github.com/your-repo/Comfyui-WL-MainImageDesign.git
```

## 🎮 使用说明

| 节点名称 | 功能描述 |
|---------|---------|

| 参数 | 说明 | 默认值 |
|-----|------|-------|
| `api_url` | LLM API 地址 | https://api.openai.com/v1 |
| `api_key` | API 密钥 | - |
| `model_name` | 模型名称 | gemini-2.0-flash-exp |
| `product_type` | 产品类型 | 摩托车头盔 |
| `selling_points` | 核心卖点 | - |

| `scene_mode` | 场景模式 | 3种可选 |
| `aspect_ratio` | 画幅比例 | 1:1 / 3:4 |
| `prompt_count` | 生成数量 | 1-10 |
| `output_language` | 输出语言 | 中文/英文 |

## 🎨 9种专业设计风格

### 1. 专业机能风
- **场景**：专业车库/维修间环境
- **色调**：深灰 #2d2d2d 或冷白 #f5f5f5
- **适用**：功能性装备、通讯设备、骑行配件

### 2. 硬核竞技风
- **场景**：赛道压弯、维修区、高速骑行
- **色调**：深黑 #0a0a0a，红/橙激情点缀
- **适用**：赛车头盔、赛道皮衣、高性能装备

### 3. 工业机械风
- **场景**：重型机械车间、改装车库
- **色调**：深灰金属 #3d3d3d，工业蓝
- **适用**：金属护具、机械配件、工具类产品

### 4. 都市通勤风
- **场景**：城市街道、咖啡店、写字楼停车场
- **色调**：自然城市色彩，灰调带暖色
- **适用**：通勤头盔、日常骑行服、都市配件

### 5. 户外冒险风
- **场景**：山路骑行、荒野休息、越野穿越
- **色调**：大地色系（土色、军绿、深蓝）
- **适用**：ADV头盔、防水装备、长途旅行用品

### 6. 高端质感风
- **场景**：高端展厅、私人车库、极简摄影棚
- **色调**：深色渐变 #1a1a1a→#2d2d2d
- **适用**：高端头盔、品牌皮具、限量款装备

### 7. 安全守护风
- **场景**：安全测试、家人送别、专业培训
- **色调**：稳重蓝 #1e3a5f 或沉稳灰
- **适用**：安全头盔、护具、新手友好装备

### 8. 改装美学风
- **场景**：改装车间、潮流街拍、霓虹夜景
- **色调**：霓虹粉/蓝/绿，大胆对比
- **适用**：彩绘头盔、改装配件、个性装备

### 9. 参数对比风
- **场景**：干净对比环境、实验室测试
- **色调**：中性白 #ffffff 或浅灰 #f5f5f5
- **适用**：功能性产品、需要参数对比的装备

## 🛡️ 质量保障

### 主标题规范
- **字号**：52-60pt 超大加粗，绝对视觉焦点
- **字重**：900-950，笔画饱满厚实
- **对比度**：与背景至少 8:1
- **位置**：画面上 15-20% 区域

### 人体完整性约束 ⚠️
当场景涉及人物/骑士时，严格保证：
- ✅ 手指 5 根完整，关节自然
- ✅ 手臂比例协调，无断裂变形
- ✅ 姿态符合物理规律
- ✅ 穿戴逻辑正确

**降低出错建议**：
- 优先侧面/背影/剪影
- 手部被产品遮挡（如握把手套只露手背）
- 远景人物（占比 <30%）
- 局部特写（手腕以下握住产品）

### 安全认证标识
支持行业标准认证展示：
- ECE（欧洲标准）
- DOT（美国交通部）
- SNELL（斯奈尔基金会）
- 3C（中国强制认证）
